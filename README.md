# Ionic Practice Projects Archive

This repository contains components, pages, experiments, and implementation examples created while learning and practicing with Ionic and Angular.

The original project accumulated outdated dependencies and legacy configurations over time, so it was simplified to preserve only the source files and project structure as a development reference.

## Repository Purpose

This project is no longer intended to be built or deployed as a complete application.

Instead, it now serves as a personal reference archive containing reusable examples, UI patterns, integrations, and experimental features that may be useful in future Ionic or Angular projects.

## Project Structure

The repository includes multiple practice modules and feature experiments such as:

### Feature Pages

- `admin-formulario` — Form management examples
- `admin-respuestas` — Response administration views
- `agenda-de-contactos` — Contact management functionality
- `apitester` — API request and testing examples
- `calculadora` — Calculator implementation
- `convertidor-de-monedas` — Currency converter example
- `conwaygame` — Conway’s Game of Life implementation
- `home` — Main landing/home page
- `infinitescrolla` — Infinite scroll implementation examples
- `login` / `loginpage` — Authentication and login UI examples
- `simple-algorithms` — Small algorithm and logic experiments

### Shared Architecture

- `components` — Reusable Ionic/Angular UI components
- `services` — Shared services, API logic, and utilities
- `assets` — Images, icons, and static resources

### Additional Examples

The project may also contain:

- Firebase integration examples
- Plotly chart implementations
- Routing/navigation patterns
- Form validation examples
- API integration logic
- State handling experiments
- Mobile-oriented Ionic UI patterns

## Project Status

⚠️ Archived Reference Project

This repository is no longer actively maintained as a working Ionic application.

The following were intentionally removed or excluded:

- Old dependency configurations
- Build environments
- Legacy tooling
- Capacitor/Ionic setup files
- Package maintenance
- Deployment configurations

The remaining code is preserved only for educational and reference purposes.


## Recommended Usage

If you want to reuse any feature or component from this repository:

1. Create a new Ionic project using the latest supported versions
2. Copy only the required pages/components/services
3. Rebuild the functionality incrementally
4. Avoid reusing old project configurations directly

## Notes

Because this repository originated as a long-term practice project, some sections may:

- use older Angular/Ionic patterns
- contain experimental code
- include incomplete implementations
- follow inconsistent architecture styles

Despite that, the repository can still be useful as a source of implementation references and reusable ideas for future projects.






# Archivo de Proyectos de Práctica con Ionic

Este repositorio contiene componentes, páginas, experimentos y ejemplos de implementación creados durante prácticas y aprendizaje con Ionic y Angular.

El proyecto original acumuló dependencias desactualizadas y configuraciones legacy con el tiempo, por lo que fue simplificado para conservar únicamente los archivos fuente y la estructura del proyecto como referencia de desarrollo.

## Propósito del Repositorio

Este proyecto ya no está pensado para compilarse o desplegarse como una aplicación completa.

Ahora funciona como un archivo de referencia personal que contiene ejemplos reutilizables, patrones de UI, integraciones y funcionalidades experimentales que pueden ser útiles en futuros proyectos de Ionic o Angular.
